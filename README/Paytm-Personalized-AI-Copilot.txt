## â–¶ï¸ How to Run the Project

1) Start Ollama (Required)

Make sure Ollama is installed and pull a model:

ollama pull phi3

(Ollama runs automatically in the background on Windows)

---

2) Start AI Service

uvicorn ai_service.main:app --reload --port 8000

Check:
http://localhost:8000/docs

---

3) Start Backend

cd backend
npm install
node index.js

Backend runs on:
http://localhost:5000

---

4) Start Frontend

cd frontend
npm install
npm run dev

Frontend runs on:
http://localhost:8080
